# Digital Portfolio Management System

## Project Overview

A PHP-based web application developed for the Open Source Technologies (CP 222) course. The system allows users to register, log in, manage digital portfolio projects, view portfolio entries, and search projects.

## Degree Program

Bachelor of Science in Multimedia Technology and Animation (MTA)

## Group Number

07

## Features

- User Registration
- User Login and Logout
- Add Portfolio Items
- View Portfolio Entries
- Search Portfolio Items
- Portfolio Summary Dashboard

## Installation Steps

1. Install XAMPP.
2. Start Apache and MySQL.
3. Copy the project folder into `htdocs`.
4. Import `database.sql` into phpMyAdmin.
5. Open the project in your browser:

   http://localhost/OpenSource_Assignment_MTA_Group_07/
